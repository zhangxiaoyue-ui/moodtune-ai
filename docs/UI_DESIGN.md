# MoodTune AI — UI 设计文档

> 技术：Streamlit + 自定义 CSS | 风格：Spotify 深色沉浸式

## 一、设计系统

### 配色

| 用途 | 颜色 | 说明 |
|------|------|------|
| 页面背景 | `#050508` → `#0d0d12` → `#1a1a2e` | 三层径向渐变 |
| 侧边栏 | `#0f0f14` → `#16161f` | 线性渐变 |
| 顶栏 | `#0a0a0f` | 纯色 + 底部微光边框 |
| 品牌主色 | `#1ed760`（Spotify 绿） | 徽章、边框、高亮 |
| 标题渐变 | `#1ed760` → `#5eead4` → `#b967ff` | 绿→青→紫 |
| 卡片背景 | `rgba(28,28,36,0.95)` | 深色玻璃质感 |
| 文字主色 | `#ffffff` / `#f5f5f7` | 纯白/近白 |
| 文字辅色 | `#9ca3af` / `#a1a1aa` | 灰色 |
| 封面渐变 1 | `#1db954` → `#169c46` | 绿色（Track 1） |
| 封面渐变 2 | `#b967ff` → `#7c3aed` | 紫色（Track 2） |
| 封面渐变 3 | `#ff6b6b` → `#ee5a24` | 红色（Track 3） |
| 网易云按钮 | `#e91429` → `#c8102e` | 红色渐变 |
| QQ 音乐按钮 | `#31c27c` → `#12b35f` | 绿色渐变 |

### 字体

- 系统默认字体
- 标题：2.35rem / 800 字重 / -0.03em 字间距
- 歌名：1.45rem / 800 字重
- 歌手：0.95rem / 500 字重
- 推荐理由：0.92rem / italic

### 间距与圆角

- 内容区最大宽度：820px
- 卡片圆角：16px
- Hero 底圆角：28px
- 按钮圆角：10px
- 标签圆角：999px（全圆角 pill）

## 三、组件详情

### Hero Banner

- 全宽渐变背景，超出内容区 margin
- 径向渐变光晕：绿色在顶部中心、紫色在右上角
- `::before` 伪元素：右上角 CD 碟片装饰（conic-gradient 同心圆）
- `::after` 伪元素：底部绿色发光分割线
- 内容居中：绿色品牌徽章 pill → 大字标题 → 灰色副标题 → 三枚功能标签

### 品牌徽章

- 内联 flex、绿色半透明底 + 绿色边框
- 文字：`🎵 MOODTUNE · AI PLAYLIST`，绿色、0.78rem、700 字重、大写间距

### 标题

- 白色 `MoodTune` + 渐变 `AI`（`-webkit-background-clip: text`）
- 2.35rem、800 字重

### 功能标签

- 半透明背景 + 细边框
- 内容："🧠 情绪理解"、"🎧 3 首精选"、"⚡ 秒级推荐"

### 心情输入区

- 情绪 Pills（Streamlit `st.pills`）：7 种情绪多选（高兴/焦虑/疲惫/浪漫/怀旧/平静/愤怒）
- 场景下拉框：不限/通勤/运动/睡前/学习/工作
- 能量滑块：1-10
- 自由文本区：100px 高度，placeholder 提示

### 歌单区域标题

- 渐变背景条（绿色→紫色方向）
- 左侧 CSS 黑胶唱片：56px 圆，conic-gradient 纹理 + inset 阴影模拟唱片槽
- 右侧：h2 标题 + 灰色说明文字

### 歌曲卡片

- Streamlit `st.container(border=True)` 包裹
- 深色玻璃质感背景 + 16px 圆角 + 阴影
- Hover 效果：边框变绿 + 绿色辉光阴影
- 内部双列布局：左侧 17% 封面区 + 右侧 83% 信息区

### 专辑封面

- 优先显示 Deezer API 真实封面（120×120px、12px 圆角、阴影）
- 降级为纯色渐变方块 + emoji 居中

### 歌曲信息

- `TRACK 01` 编号标签：小号灰色大写
- 歌名：白色 1.45rem 粗体，🎵 前缀
- 歌手：灰色 0.95rem，🎤 前缀
- 推荐理由：左边 3px 绿色竖线 blockquote、绿色半透明底、italic、"AI ·" 前缀

### 收听按钮

- flex 等宽排列，顶部细分隔线
- 网易云：红色渐变底 + 白色字 + 红阴影
- QQ 音乐：绿色渐变底 + 深色字 + 绿阴影
- Hover：上移 1px

### 点赞踩反馈

- 每首歌底部 👍 👎 按钮
- 点赞后显示绿色 "已点赞" badge
- 点踩后显示红色 "已点踩" badge

### 一键微调按钮行

- 🔥 更激昂 / 🌙 更安静 / 🔄 换一批 / ➕ 再推几首
- 点踩和微调会累积为下轮 Prompt 上下文

### 反馈面板

- 深色卡片，顶部细分隔线
- 整体满意度 1-5 星 + 可选文字评论
- 提交后显示感谢信息

### 分享卡片

- PIL 渲染 PNG 图片
- 包含歌单信息、QR 码
- 下载按钮

## 四、交互状态

| 状态 | 表现 |
|------|------|
| 初始 | Hero + 空输入区 + 空结果区 |
| 输入中 | 实时表单交互，无阻塞 |
| 生成中 | spinner + 流式输出文字 |
| 成功 | 3 首歌曲卡片 + 微调按钮 + 反馈面板 |
| API 错误 | 红色错误提示 + 重试建议 |
| 空结果 | "没有生成歌曲，请重试" |

## 五、动画与过渡

| 元素 | 效果 |
|------|------|
| 歌曲卡片 | border-color 0.2s + box-shadow 0.2s hover |
| 收听按钮 | transform 0.15s + box-shadow 0.15s + opacity 0.15s hover |

动画风格克制，仅 hover 微交互，无入场动画。
